    static, constant isWebRTCSupported :Boolean

    当前浏览器是否支持 WebRTC 的标记

    static, constant WebRTCPlugin

    WebRTC 插件，使用后可通过 Realtime#createWebRTCClient 创建 WebRTCClient 实现音视频通话
    Example
    var realtime = new Realtime({
  appId: appId,
  appKey: appKey,
  server: server,
  plugins: WebRTCPlugin,
});
realtime.createWebRTCClient(id);
